# Muhammad Qasim Portfolio

A modern personal portfolio website with smooth scrolling and animation.

## Project Structure

- index.html
- assets/
  - css/styles.css
  - js/main.js
  - vendor/
    - gsap.min.js
    - ScrollTrigger.min.js
    - lenis.min.js
    - three.min.js
- legacy/
  - css2.old.css
  - index.old.html
  - script.old.js
  - styles.old.css

## Run Locally

1. Open a terminal in this folder.
2. Run: npm run dev
3. Open the shown local URL in your browser.

## Customize Quickly

- Change your text content in index.html.
- Update colors and typography in assets/css/styles.css.
- Adjust animation timing in assets/js/main.js.
